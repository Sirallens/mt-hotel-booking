# MT Hotel Booking System

Plugin de cotización y solicitud de reservaciones para WordPress. Estructura modular lista para desarrollo.

## Estructura

- hotel-booking-system.php — Archivo principal (bootstrap)
- admin/ — Lógica y pantallas de administración
- public/ — Hooks y vistas del frontend
- includes/ — Clases de lógica y helpers
- assets/ — CSS y JS
- languages/ — Traducciones

## Instalación
1. Copia la carpeta en `wp-content/plugins/`
2. Activa el plugin desde el panel de WordPress
